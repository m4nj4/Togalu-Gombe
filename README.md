# 🎭 Togalu-Gombe (ತೊಗಲು ಗೊಂಬೆ)
### Digital Shadow-Theater Companion App

> *Bringing Karnataka's ancient Leather Puppetry art to the digital generation.*

---

## 📖 About the Project

**Togalu-Gombe** is an Android app built as a digital companion for Karnataka's traditional **Togalu Gombeyaata** (Leather Shadow Puppetry) — one of India's oldest visual art forms.

The app bridges the **generation gap** by providing real-time scene summaries, an interactive puppet gallery, and a platform to connect with local artists — making this ancient art accessible and exciting for today's youth.

---

## ❗ Problem Statement

Leather Puppetry (Togalu Gombeyaata) is one of Karnataka's oldest visual arts, but its stories are narrated in **Old Kannada** which the younger generation finds hard to follow. The art is rapidly losing its entertainment value due to the language and pace gap, risking the loss of this rich cultural heritage forever.

**Togalu-Gombe** solves this by acting as a smart digital companion that provides scene guides, character information, and artist connect — right from your Android phone.

---

## ✨ Features

### 🎬 Live Assist
- Select a play (Ramayana, Mahabharata, Harishchandra)
- Get scene-by-scene breakdown in simple English
- Swipe through scenes in sync with the live performance

### 🪆 Puppet Gallery
- Interactive grid of puppet characters
- Zoomable images showing intricate leather perforations
- Character symbolism, role, and backstory for each puppet

### 🎨 Artist Connect
- Book puppet-making workshops with local artists
- Buy authentic miniature puppets as home decor
- Watch short history videos on leather curing and dyeing

## 🚀 Getting Started

### Prerequisites
- Android Studio (latest)
- Android SDK API 21+
- Java 8+

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/Togalu-Gombe.git
```

2. **Open in Android Studio**
```
File → Open → Select the Togalu-Gombe folder
```

3. **Build & Run**
```
Click Run ▶️ or press Shift + F10
```
